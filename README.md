**Casino games**
-------------------------------------------
Набор мини-игр запускаемых из консоли. На выбор представлены такие игры как:
 - однорукий бандит
 - пьяница
 - очко 

**Требования для запуска:**
  - Java 11
  
**Стек технологий:**
 - Java 

**Для запуска:**
 ```
$ make #build & run
```
